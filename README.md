# Javascrip Drawing Canvas 프로젝트
https://oz528.github.io/canvas/

그림을 그릴 수 있는 캔버스에 이미지 삽입과 드로잉, 저장을 구현한 프로젝트입니다.

## 구현 내용
- Javascript를 이용해 그림판의 기능을 구현했습니다.
- 브러쉬의 색상 변경/크기 조절, 이미지 불러오기/저장하기 등이 가능합니다.
- Javascript 위주의 프로젝트입니다.
